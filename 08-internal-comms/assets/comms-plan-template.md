# comms-plan-template.md — Multi-Channel Communication Plan

## What This Is
A planning template for coordinating internal communications across multiple channels and audiences for a significant change or announcement. The skill uses a completed version to help sequence and draft all the communications needed, not just the main announcement.

## What To Add
Fill this in for any change that requires more than one message — different audiences, multiple channels, or staged communication. The skill will help you draft each piece once the plan is mapped.

# talking-points-template.md — Manager Cascade Template

## What This Is
A template for preparing managers to have team-level conversations about an organizational change. The skill uses a completed version of this to generate talking points that managers can use in 1:1s or team meetings — consistent messaging, but not a script.

## What To Add
Fill this out for any change where managers need to deliver the message to their teams, not just forward an email. The skill will help draft the actual talking points once you've filled in the context.

## When To Use This
- Restructuring or role changes affecting specific teams
- Difficult news (budget cuts, layoffs, program discontinuation)
- Policy changes requiring manager explanation
- Leadership transitions where teams need reassurance

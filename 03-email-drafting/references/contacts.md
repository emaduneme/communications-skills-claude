# contacts.md — Optional Reference File

## What This Is
A reference list of key contacts with relationship context. This is entirely optional, but if you frequently email the same people, adding context here means you don't have to re-explain the relationship every time.

## What To Add
For each person you email regularly, add:
- Name and title
- Your relationship (supervisor, peer, student, collaborator, external contact)
- Formality level (very formal / professional / collegial / casual)
- Any preferences or quirks (e.g., "prefers short emails," "responds better to bullet points," "very formal, always use title")

## Format
```
## [Name] — [Title, Organization]
- Relationship: [how you know them / power dynamic]
- Formality: [very formal / professional / collegial / casual]
- Notes: [anything useful]

## Example:
## Dr. Sarah Chen — Department Chair, Journalism
- Relationship: My direct supervisor
- Formality: Professional (warm but formal)
- Notes: Prefers brief emails. Always use "Dr. Chen." Responds quickly to bullet-pointed asks.

## Marcus Williams — PhD student, my advisee
- Relationship: Mentee, writing under my supervision
- Formality: Collegial/supportive
- Notes: First-gen student, can be anxious about feedback. Frame corrections constructively.
```

# voice-samples.md — Reference File

## What This Is
2–3 cover letters you've written that you're proud of and consider "on voice." The skill analyzes these to match your tone, rhythm, and phrasing when generating new letters.

## What To Add
Paste 2–3 complete cover letters you've previously written. These should:
- Be letters you're happy with (don't include ones you felt missed the mark)
- Cover different types of roles if possible (e.g., one academic, one industry)
- Be as close to what you actually sent as possible

## Why This Matters
Everyone writes differently. Some people use short, punchy sentences. Others write in flowing, subordinate clauses. Some lead with personality; others lead with credentials. Without samples, the skill defaults to a neutral professional voice — which works fine but may not sound like *you*.

## Format
Just paste them in as plain text, separated by a divider like `---`.

## Example Structure
```
--- SAMPLE 1: Application for Assistant Professor, [University], [Year] ---

Dear Search Committee,

[Full letter text]

Warmest Regards,
[Name]

---

--- SAMPLE 2: Application for UX Researcher, [Company], [Year] ---

[Full letter text]

---
```

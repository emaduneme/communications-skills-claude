# Brand Voice — Your Reference Materials

This folder is for brand materials **you provide**. Add files here to get more tailored output from the brand-voice skill.

---

## What to put here

| File | What it contains | Why it helps |
|------|-----------------|--------------|
| `existing-guidelines.md` | Any brand or style guidelines your org already has | Lets Claude extend rather than start from scratch |
| `sample-communications.md` | 3–5 examples of writing you consider on-brand | Gives Claude concrete models to match |
| `terminology.md` | Preferred terms, avoided words, proper nouns | Prevents terminology inconsistencies in the output |
| `audience-notes.md` | Notes on who you're writing for | Keeps voice calibration accurate to your actual readers |

None of these files are required — the skill works without them. But each one you add makes the output more specific to your organization.

---

## How to add a file

1. Create a new `.md` file in this folder (name it anything descriptive)
2. Paste or type your content
3. Attach it to your Claude project alongside the `SKILL.md` instructions

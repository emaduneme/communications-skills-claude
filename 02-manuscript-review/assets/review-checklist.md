# Manuscript Self-Review Checklist

Use this before running the manuscript-review skill or before submission. Work through each section independently, saving the skill for deeper citation and statistical verification.

---

## Structural Consistency

- [ ] Abstract sample sizes match body text
- [ ] All hypotheses stated in the introduction are addressed in the results
- [ ] Hypothesis labels (H1, H2a, H2b...) are consistent throughout
- [ ] Methods section describes every analysis that appears in Results
- [ ] All numbers cited in prose match corresponding tables and figures
- [ ] Introduction claims are backed by citations

---

## Citation Spot-Check

- [ ] Author names in text match reference list (spelling and order)
- [ ] Publication years in text match reference list
- [ ] "Et al." only used for 3+ authors (APA 7th)
- [ ] DOIs present for all journal articles
- [ ] No placeholder citations left (e.g., "Author, year")

---

## Statistical Reporting

- [ ] Degrees of freedom present for every test statistic
- [ ] p-values include direction (p < .05, not p = .05)
- [ ] Confidence intervals formatted: B = X.XX, 95% CI [LL, UL]
- [ ] Effect sizes reported where expected
- [ ] Sample sizes consistent across all mentions
- [ ] Decimal places consistent (2 for statistics, 3 for p-values)

---

## Copy-Paste and Carryover Errors

- [ ] No topic contamination (wrong condition or study language)
- [ ] No leftover placeholder or template text
- [ ] Terminology for key constructs is consistent throughout
- [ ] For multi-study papers: no Study 1 language in Study 2 sections
- [ ] Author-identifying information removed if blind review

---

## Writing and Style

- [ ] Each paragraph has a clear topic sentence
- [ ] Technical terms defined on first use
- [ ] Active voice dominant (passive acceptable in Methods)
- [ ] No hedging beyond what is scientifically appropriate
- [ ] Section transitions are logical and explicit

---

## Before Submission

- [ ] Word count is within journal guidelines
- [ ] Figures and tables are numbered sequentially and titled
- [ ] Supplementary materials referenced correctly in text
- [ ] Acknowledgments, funding, and conflict of interest statements are current
- [ ] Cover letter addresses any prior reviewer feedback

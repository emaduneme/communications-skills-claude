# common-errors.md — Reference File

## What This Is
A running log of recurring errors caught in your previous manuscript reviews. Over time, this file trains the skill to look for your specific patterns — not just generic issues.

## What To Add
Document errors you've seen repeatedly in your own work or your students' work. Group them by category.

## Format

```
## Citation Errors
- I frequently confuse [Author A] and [Author B] — always double-check
- My co-author's name is sometimes listed wrong: correct spelling is [X]
- I tend to use outdated editions of [style guide] — current is [version]

## Statistical Reporting
- I sometimes forget to report degrees of freedom for F-tests
- My lab uses [software] which outputs [format] — check against APA 7th
- I often round p-values incorrectly when they're < .001

## Copy-Paste Artifacts
- Multi-study papers: watch for Study 1 stimuli descriptions bleeding into Study 2
- I reuse intro paragraphs from grants — check for funding-specific language

## Style Issues
- I overuse em dashes — flag any
- I tend to bury the hypothesis late in the intro — check structure
- My methods sections default to passive voice throughout — flag any passive in discussion
```

## How to Use
Update this file after each review session. The skill will check for these patterns first before doing a general review.
